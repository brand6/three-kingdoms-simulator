## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Inline delta chips, compact label-to-value gaps |
| sm | 8px | Button gaps, selector row spacing, compact metadata stacks |
| md | 16px | Default panel padding, task card padding, dialog inner margins |
| lg | 24px | Task card section spacing, main monthly report section gaps |
| xl | 32px | Scene edge margins, top bar spacing, major three-column layout gaps |
| 2xl | 48px | Minimum action button height, major dialog breathing room |
| 3xl | 64px | Top-level popup/report vertical rhythm only |

Exceptions: 12px is allowed for dense stacked content inside task cards, month report sections, and selector/profile popups; 72px bottom navigation height remains as the persistent HUD anchor.

Source notes: existing `MainScene.tscn` already uses 32px outer margins, 24px main content gap, 16px card gaps, 8px button gaps, and 48px button heights; this spec preserves that rhythm for month-start and month-end overlays.

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #F5F0E6 | Full-scene background, dialog canvas, non-interactive report surfaces |
| Secondary (30%) | #E4D8C5 | Cards, sidebar panels, popup panels, task summary rail, neutral buttons |
| Accent (10%) | #2F5D62 | Primary CTA, selected task card border, current task progress, promotion success header, key positive delta badges |
| Destructive | #B42318 | End-xun confirmation, irreversible warning text, failed task / unmet-threshold alerts only |

Accent reserved for: `领取主任务` primary button, currently selected monthly task card, task progress number or progress bar, promotion success title, and positive month-end summary badges. Do not use accent on every nav button or every clickable row.

Source notes: colors come from `PrototypeTheme.tres` (`PageBackground`, `PanelSurface`, `Accent`, `Destructive`). Phase 02.1 context adds accent-worthy moments at task lock-in, visible progress, and promotion outcome.

## Copywriting Contract

| Element | Copy |
|---------|------|
| Primary CTA | 领取主任务 |
| Empty state heading | 本月暂无可领任务 |

Source notes: `领取主任务` and task lock wording come from 02.1 context D-01/D-03/D-04; end-xun confirmation follows existing `MainHUD.gd` copy style; empty and error text are prescriptive defaults for this phase because upstream artifacts defined behavior but not exact strings.
